Named after Kahneman’s System 2: analytical, deliberate, and risk-aware—not fast intuition alone.

Philosophy

Intuition is fast. Engineering should be slow.

Agent-assisted coding defaults to System 1 thinking: quick guesses, implicit requirements, and “looks done” diffs. Deliberate Code tools encode System 2 habits—explicit scope, written specs, and checkpoints before merge.

System 1

Fast, automatic, pattern-matching. Great for exploration; risky as the only mode for production changes.

System 2

Slow, effortful, analytical. The mode our tools reinforce—through specs, gates, and verification built into the workflow.
